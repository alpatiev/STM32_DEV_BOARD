# RESOURCES INFO

The "RESOURCES" directory includes the following items:

1. Generated files: files that are generated as part of the project build process. 
These files may include binary executables, object files etc.

2. Project assets: project-specific assets or multimedia files. 
These could include images, audio files, or any other assets that are used within the project.

3. Documentation files: this directory is also dedicated to storing documentation related to the project. 
It may include user manuals, API documentation, design documents, or any or implementation details.
